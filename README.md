# Класифікація інтентів україномовних звернень

Навчальний NLP-проєкт для аналізу інтентів в україномовних користувацьких повідомленнях. У проєкті є ноутбук з EDA, підготовкою ознак, навчанням моделей і оцінюванням результатів, а також Gradio-демо для швидкої перевірки збереженої моделі.

## Вміст проєкту

- `EDA + моделі.ipynb` — дослідницький аналіз даних, препроцесинг, навчання та порівняння моделей.
- `gradio demo.py` — демонстраційний інтерфейс на Gradio для класифікації нового текстового звернення.
- `models/intent_logreg_tfidf.joblib` — збережена модель `TF-IDF + Logistic Regression`.
- `models/model_metadata.json` — метадані моделі та основні метрики.
- `requirements.txt` — мінімальний перелік залежностей для запуску.

## Структура

```text
project/
├── EDA + моделі.ipynb
├── gradio demo.py
├── models/
│   ├── intent_logreg_tfidf.joblib

```

## Необхідні залежності

Встановлення через `requirements.txt`:

```bash
pip install -r requirements.txt
```

Або вручну:

```bash
pip install pandas numpy matplotlib scikit-learn joblib gradio notebook
```

## Запуск ноутбука

1. Створіть і активуйте віртуальне середовище Python.
2. Встановіть залежності.
3. Переконайтеся, що файл `UAReviews.csv` доступний у робочій директорії ноутбука або змініть шлях до нього в коді.
4. Запустіть Jupyter:

```bash
jupyter notebook
```

5. Відкрийте `EDA + моделі.ipynb` і виконуйте клітинки послідовно.

## Запуск Gradio-демо

1. Переконайтеся, що файл `models/intent_logreg_tfidf.joblib` присутній у папці `models/`.
2. Запустіть застосунок:

```bash
python "gradio demo.py"
```

3. Після запуску відкрийте локальне посилання Gradio в браузері та введіть україномовне звернення для класифікації.

## Що робить демо

Gradio-інтерфейс:

- очищує вхідний текст;
- перетворює його у TF-IDF-вектор;
- визначає передбачений інтент;
- показує `confidence`;
- виводить імовірності для всіх класів.

## Примітка

У цій папці зараз немає `UAReviews.csv`, тому ноутбук у поточному вигляді потребує або наявності датасету поруч, або оновлення шляху до файлу. Для Gradio-демо датасет не потрібен, якщо збережена модель уже існує.
